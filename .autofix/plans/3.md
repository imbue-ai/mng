# Issue 3: Missing underscores in gc_test.py test function names

## What
In `libs/mngr/imbue/mngr/cli/gc_test.py`, lines 34, 42, 50, 57, 64, and 427, the test
functions were renamed from `test_format_size_*` to `testformat_size_*` (missing the
underscore between `test` and `format_size`).

## Why it is a problem
pytest discovers test functions by looking for the `test_` prefix. Without the underscore,
these 6 test functions will NOT be discovered or run by pytest. This means format_size
is effectively untested, silently.

## Planned fix
Rename all 6 functions to restore the `test_` prefix:
- `testformat_size_bytes` -> `test_format_size_bytes`
- `testformat_size_kilobytes` -> `test_format_size_kilobytes`
- `testformat_size_megabytes` -> `test_format_size_megabytes`
- `testformat_size_gigabytes` -> `test_format_size_gigabytes`
- `testformat_size_terabytes` -> `test_format_size_terabytes`
- `testformat_size_parametrized` -> `test_format_size_parametrized`

## Risks
None - straightforward rename to restore correct pytest naming convention.
